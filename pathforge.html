1 def reverse_list(lst): 2 for i in range(len(lst)): 3 lst[i], lst[len(lst) - i] = \ 4 lst[len(lst) - i], lst[i] 5 return lst
Line 3 has an off-by-one error. len(lst) - i accesses index out of bounds on the first iteration. Python list indices are zero-based, so the last element is at len(lst) - 1.
lst[i], lst[len(lst) - 1 - i] = lst[len(lst) - 1 - i], lst[i]
