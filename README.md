Pathforge — Adaptive CS Learning, Powered by AI
Pathforge is an AI-driven tutoring system designed specifically for Computer Science undergraduates at VIT University. It moves beyond static tutorials by learning how each student thinks, adapting in real-time to their unique pace and knowledge gaps.

🚀 Key Features
Deep Knowledge Tracing (DKT): The system models your learning state after every interaction to update a live knowledge map.

Adaptive Assessment: Uses Item Response Theory (IRT) to calibrate question difficulty based on your current performance.

AI Code Review: Provides line-level annotations and specific fixes rather than vague hints, grounded in actual CS documentation.

Concept Dependency Graph: Automatically sequences topics; if you struggle, Pathforge traces back to the "root" concept you missed.

Instructor Insights: A real-time dashboard for educators to monitor cohort engagement and identify at-risk students before exams.

🛠️ Tech Stack
Frontend: HTML5, CSS3, and Vanilla JavaScript.

Visuals: Powered by Three.js for an immersive, interactive 3D background and SVG-based knowledge graphs.

Typography: Plus Jakarta Sans, Fraunces, and JetBrains Mono.

Deployment: Hosted as a Static Space on Hugging Face.

📖 How It Works
Interaction: You answer a question or submit code.

Analysis: The system updates your mastery levels for concepts like Recursion, Data Structures, or Time Complexity.

Feedback: If you're stuck, the AI Concept Tutor explains the "why" behind the error.

Progression: Spaced repetition schedules your next review at the optimal moment to ensure long-term retention.
